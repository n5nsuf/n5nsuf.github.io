About this site

hanbe.click은 한 명의 개발자(HCB)가 운영하는 개인 기술 블로그·실험 사이트입니다. AI 자동화 도구(Claude·Gemini·MCP), 자동매매 시스템(KIS API 기반), 에이전트 프레임워크(LightHarness) 등을 실제로 만들어 운영하면서 얻은 가이드와 프로젝트 회고를 공유합니다.

상업 조직이나 팀이 아닌 1인 실험 공간이며, 일부 콘텐츠는 새로운 도구·접근을 시험하는 과정에서 작성됩니다. 모든 공개 글은 사람(HCB)의 검토를 거쳐 게시되며, AI 도구 보조 작성 분에는 사실 검증과 출처 인용을 적용합니다. 자세한 정책: 개인정보 처리방침.

hanbe.click is a personal technical blog and experimental site operated by a single developer (HCB). It shares guides and project retrospectives derived from actually building and running AI automation tools (Claude · Gemini · MCP), an algorithmic trading system (on KIS API), and an agent framework (LightHarness).

This is not a commercial entity or team — it is a solo experimental space, and some content arises from trying out new tools or approaches. All public posts are reviewed by a human (HCB) before publishing; AI-assisted drafts go through fact-checking and source citation. See: Privacy Policy.
