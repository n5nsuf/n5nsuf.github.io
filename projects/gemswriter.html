개요 Overview

GemsWriter는 Gemini Gems의 특화된 지식 체계와 Claude Haiku의 정교한 문장 생성 능력을 결합한 AI 글쓰기 봇입니다. 매주 평일 오전 08:30(KST)에 자동으로 실행되어 사전에 정의된 페르소나와 주제에 맞는 콘텐츠를 생성하고 텔레그램 채널로 발행합니다.

GemsWriter is an AI writing bot that combines the specialized knowledge base of Gemini Gems with the sophisticated generation capabilities of Claude Haiku. It runs automatically every weekday at 08:30 KST, generating content aligned with predefined personas and themes, and publishing it to a Telegram channel.

🤖

이 프로젝트는 다중 LLM 협업 패턴을 활용하여, 각 모델의 강점(Gemini의 지식 검색 및 Gems 지식, Claude의 문체 및 요약 능력)을 극대화합니다.

This project utilizes a multi-LLM collaboration pattern, maximizing each model's strengths: Gemini's knowledge retrieval/Gems context and Claude's stylistic/summarization capabilities.

주요 기능 Features

  • Gems 지식 활용: Gemini Gems에 구축된 특정 도메인 지식과 페르소나를 기반으로 깊이 있는 초안 생성
  • 하이브리드 생성: Gemini의 초안을 Claude Haiku가 다듬어 독자에게 친숙한 톤앤매너로 변환
  • 완전 자동화 스케줄링: GitHub Actions를 통해 추가 조작 없이 평일 아침 정해진 시간에 작동
  • 실시간 텔레그램 연동: 생성된 포스트를 즉시 텔레그램 채널로 전송하고 운영자 알림 발송
  • Gems Knowledge: Generates in-depth drafts based on specific domain knowledge and personas built in Gemini Gems
  • Hybrid Generation: Claude Haiku refines Gemini's drafts into a reader-friendly tone and manner
  • Fully Automated Scheduling: Operates at set times on weekday mornings via GitHub Actions with zero manual intervention
  • Real-time Telegram Integration: Immediately sends generated posts to Telegram channels and triggers admin notifications

아키텍처 Architecture

스케줄링 기반 하이브리드 LLM 콘텐츠 생성 워크플로우 Scheduling-based hybrid LLM content generation workflow

작동 구조 및 AI의 역할 Working Structure & AI's Role

시스템은 GitHub Actions 트리거 → Gemini 데이터 추출 → Claude 문장 가공 → 텔레그램 발행의 파이프라인으로 구성됩니다. 매일 정해진 시간에 서버리스 환경에서 구동되어 유지보수 비용을 최소화합니다.

The system consists of a pipeline: GitHub Actions Trigger → Gemini Data Extraction → Claude Refinement → Telegram Publishing. It runs in a serverless environment at scheduled times to minimize maintenance costs.

AI의 역할

The Role of AI

  • 지식 큐레이션 (Gemini): Gemini Gems에 학습된 전문 지식을 바탕으로 특정 주제에 대한 핵심 인사이트와 로데이터를 추출합니다.
  • 문체 및 페르소나 적용 (Claude): Gemini가 생성한 초안을 바탕으로, 설정된 작가 페르소나에 맞춰 독창적이고 매끄러운 문장으로 재구성합니다.
  • 품질 검수: 생성된 콘텐츠가 채널의 가이드라인과 길이를 준수하는지 스스로 판단하여 보정합니다.
  • Knowledge Curation (Gemini): Extracts key insights and raw data on specific topics based on expertise trained in Gemini Gems.
  • Style & Persona Application (Claude): Reconstructs Gemini's draft into creative and smooth sentences aligned with the set author persona.
  • Quality Control: Self-evaluates and adjusts generated content to ensure it follows channel guidelines and length requirements.
