개요 Overview

HTML Writer는 복잡한 HTML 구조를 쉽고 빠르게 생성할 수 있도록 돕는 저작 도구입니다. 미리 정의된 디자인 토큰과 레이아웃 패턴을 활용하여, 코드 작성 없이도 스타일 가이드에 부합하는 웹 페이지를 구성할 수 있습니다.

HTML Writer is an authoring tool designed to help create complex HTML structures easily and quickly. Utilizing predefined design tokens and layout patterns, users can compose web pages that adhere to style guides without writing code.

✍️

이 프로젝트는 정적 사이트 제너레이터(SSG)를 위한 전처리기로도 활용 가능하며, 일관된 UI/UX 유지를 보장합니다.

This project can also be used as a preprocessor for Static Site Generators (SSG), ensuring consistent UI/UX maintenance.

주요 기능 Features

  • 라이브 프리뷰: 작성 즉시 렌더링 결과를 확인하는 실시간 동기화
  • 섹션 템플릿: 헤더, 푸터, 카드 그리드 등 자주 쓰이는 레이아웃의 원클릭 삽입
  • 다국어 태그 자동 생성: lang-ko, lang-en 구조를 자동으로 매칭하여 삽입
  • 스타일 가이드 준수: 프로젝트별 CSS 토큰을 강제 적용하여 디자인 파편화 방지
  • Live Preview: Real-time synchronization to check rendering results as you write
  • Section Templates: One-click insertion of common layouts like headers, footers, and card grids
  • Auto Multilingual Tags: Automatically matches and inserts lang-ko and lang-en structures
  • Style Guide Compliance: Enforces project-specific CSS tokens to prevent design fragmentation

작동 구조 및 에이전틱 AI 도입 Operational Structure & Agentic AI Integration

HTML Writer는 단순한 텍스트 변환을 넘어, 여러 특화된 AI 에이전트가 협력하여 고품질의 웹 콘텐츠를 생산하는 에이전틱 워크플로우(Agentic Workflow)를 채택하고 있습니다.

Beyond simple text transformation, HTML Writer employs an Agentic Workflow where multiple specialized AI agents collaborate to produce high-quality web content.

🤖

에이전틱 AI는 단일 LLM 호출의 한계를 극복하기 위해 '계획-실행-검증'의 반복 루프를 수행하며, 각 단계에서 전문화된 페르소나를 활용합니다.

Agentic AI overcomes the limitations of single LLM calls by performing an iterative 'Plan-Act-Verify' loop, utilizing specialized personas at each stage.

1. 에이전트 팀 구성

1. Agent Team Composition

  • Planner Agent: 사용자의 의도를 분석하고 페이지의 전체 구조(Outline)와 필요한 컴포넌트 배치를 설계합니다.
  • Content Agent: 설계된 구조에 따라 본문을 작성하며, 한국어와 영어의 문맥적 일치성을 유지하며 다국어 텍스트를 생성합니다.
  • Component Agent: 텍스트의 의미에 가장 적합한 UI 패턴(Callout, Card, Badge 등)을 선택하고 HTML 코드로 매칭합니다.
  • Style Validator: 최종 결과물이 STYLE_GUIDE.md의 디자인 토큰과 레이아웃 규칙을 준수하는지 검사하고 교정합니다.
  • Planner Agent: Analyzes user intent and designs the overall page structure (outline) and component placement.
  • Content Agent: Writes the body text according to the designed structure, generating multilingual text while maintaining contextual consistency between Korean and English.
  • Component Agent: Selects the most appropriate UI patterns (Callout, Card, Badge, etc.) for the text's meaning and matches them to HTML code.
  • Style Validator: Inspects and corrects the final output to ensure compliance with the design tokens and layout rules in STYLE_GUIDE.md.

3. 도입 효과 3. Benefits of Integration

  • 일관성 보장: 제작자가 달라도 동일한 디자인 언어와 코드 구조를 유지합니다.
  • 생산성 향상: 복잡한 HTML 수동 코딩 시간을 90% 이상 단축하고 콘텐츠 기획에 집중할 수 있습니다.
  • 확장성: 새로운 컴포넌트나 스타일 규칙이 추가되어도 에이전트의 지침만 업데이트하면 즉시 적용됩니다.
    • Guaranteed Consistency: Maintains the same design language and code structure regardless of the creator.
    • Increased Productivity: Reduces manual HTML coding time by over 90%, allowing focus on content planning.
    • Scalability: New components or style rules can be applied immediately by simply updating the agent's instructions.
