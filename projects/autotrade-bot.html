AI-powered Korean equities auto-trading bot — a live system combining LightGBM and LSTM
2026-04-17 프로젝트 Project 실전 운영 중 Live in Production

개요 Overview

한국투자증권(KIS) Open API를 통해 실계좌에서 운영 중인 AI 기반 주식 자동매매 봇입니다. LightGBM으로 단기 방향성을 예측하고, LSTM으로 시계열 패턴을 보조 판단하여 매수·매도 시그널을 산출합니다.

A production auto-trading bot running on a live brokerage account via the Korea Investment & Securities (KIS) Open API. LightGBM predicts short-term directionality and an LSTM model provides sequential pattern support, together producing buy/sell signals.

설계 목표는 예측 정확도 자체보다 리스크 관리와 안정적 운영입니다. 포지션 사이즈, 손절·익절 규칙, 장애 복구 로직이 모델과 동등한 비중으로 다뤄집니다.

The design goal is not peak prediction accuracy but risk management and operational stability. Position sizing, stop-loss / take-profit rules, and fault-recovery logic are treated with the same weight as the model itself.

주요 기능 Features

  • KIS Open API 기반 실시간 시세 수집 및 실계좌 주문 실행
  • LightGBM + LSTM 하이브리드 모델로 매수·매도 시그널 산출
  • 종목별 포지션 사이즈 계산 및 분산 배분
  • 손절(stop-loss), 익절(take-profit), 일일 최대 손실 한도 자동 적용
  • 장 시작·마감 스케줄러 및 비정상 종료 시 포지션 복구 로직
  • 체결·손익·에러 로그의 구조화된 기록으로 사후 분석 가능
  • Real-time quote ingestion and live-account order execution via KIS Open API
  • Hybrid LightGBM + LSTM signal engine for buy/sell decisions
  • Per-ticker position sizing and diversified allocation
  • Automated stop-loss, take-profit, and daily max-loss guardrails
  • Market open/close scheduler with position recovery after abnormal shutdowns
  • Structured logs for fills, P&L, and errors to enable post-hoc analysis

작동 구조 및 AI의 역할 Working Structure & AI's Role

시스템은 수집 → 피처 엔지니어링 → AI 추론 → 시그널 결정 → 리스크 체크 → 주문 실행의 파이프라인으로 작동합니다. 각 단계는 독립 모듈로 구성되어 장애 발생 시 해당 지점부터 복구가 가능합니다.

The system operates through a pipeline: Ingestion → Feature Engineering → AI Inference → Signal Decision → Risk Check → Order Execution. Each stage is modularized to allow recovery from the point of failure.

AI의 역할

The Role of AI

  • 수익률 예측: LightGBM을 사용하여 수백 개의 기술적 지표와 수급 데이터를 바탕으로 차기 캔들의 방향성을 분류합니다.
  • 패턴 인식: LSTM 시계열 모델이 가격의 흐름과 변동성 패턴을 분석하여 모델의 판단에 가중치를 부여합니다.
  • 동적 임계값: 시장 상황에 따라 매수·매도 시그널의 강도를 조절하여 불필요한 거래를 억제합니다.
  • Return Prediction: Uses LightGBM to classify the direction of the next candle based on hundreds of technical indicators and supply/demand data.
  • Pattern Recognition: The LSTM time-series model analyzes price flow and volatility patterns to weight the model's decisions.
  • Dynamic Thresholding: Adjusts the strength of buy/sell signals according to market conditions to suppress unnecessary trades.
⚠️

실계좌 운영 봇은 예측 품질만큼이나 장애 내성이 중요합니다. 네트워크 끊김, API 레이트리밋, 부분 체결 등 모든 예외 경로가 테스트되지 않으면 가동 금지.

For a live-account bot, fault tolerance matters as much as predictive quality. Do not run unless every edge path — network drops, API rate limits, partial fills — has been tested.
