개요 Overview

DevProfile Admin은 본 포트폴리오 사이트의 콘텐츠를 관리하고 업데이트하는 핵심 엔진입니다. 단순한 CMS를 넘어, 사용자의 명령을 이해하고 스타일 가이드를 준수하며 문서를 생성하는 에이전틱 AI 워크플로우를 제공합니다.

DevProfile Admin is the core engine for managing and updating the content of this portfolio site. Beyond a simple CMS, it provides an agentic AI workflow that understands user commands and generates documents while adhering to style guides.

스택 Stack

기술Tech 역할Purpose

주요 기능 Features

  • Headless 에디팅: GitHub Actions를 통해 로컬 환경 없이도 브라우저나 API 호출만으로 사이트 수정
  • 디자인 시스템 가드레일: 모든 생성물이 STYLE_GUIDE.md를 100% 준수하도록 강제
  • 다국어 페어 생성: 한국어 입력 시 문맥에 맞는 자연스러운 영어 번역본 동시 생성
  • AI 답변 표시 공간 (Workspace): 에이전트의 사고 과정과 작업 결과물을 실시간으로 모니터링
  • Headless Editing: Modify the site via browser or API calls using GitHub Actions, without a local environment
  • Design System Guardrails: Enforces 100% compliance with STYLE_GUIDE.md for all generated content
  • Multilingual Pair Generation: Automatically generates natural English translations for Korean inputs
  • AI Response Workspace: Real-time monitoring of the agent's reasoning process and work output

AI 답변 표시 공간 (Workspace) AI Response Workspace

관리자 인터페이스의 핵심 요소로, AI 에이전트가 사용자의 요청을 처리하는 과정을 투명하게 보여줍니다. 이는 '블랙박스' 형태의 AI 동작을 지양하고, 사용자가 AI의 계획을 검증하고 수정할 수 있는 신뢰 환경을 구축합니다.

As a core element of the admin interface, it transparently shows the process of AI agents handling user requests. This avoids 'black box' AI behavior and builds a trusted environment where users can verify and modify AI plans.
